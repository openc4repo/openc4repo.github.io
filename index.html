Open C4 Repo

This is a community project serving as both a knowledgebase as well as an index of drivers and tools used to maintain and create killer Control4 setups.
Anyone can contribute by simply forking this repository to their own Github account and sending a pull request with their changes/additions.

The repository can be found here https://github.com/openc4repo/openc4repo.github.io.
